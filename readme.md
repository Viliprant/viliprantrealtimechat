# MySlime - Real-time chat

[Link to website](https://viliprantrealtimechat.herokuapp.com/)

## Installation
Install dependencies :
````
    npm install
````

Get in the repository, then run this command:
````
    npm start
````

## Technologies:

- Vanilla Javascript
- Node.js
- Express.js
- Socket.IO
- Material icons
